# siegelmodularforms
**A Pari/GP Package for computing Siegel Modular forms for $$Sp(g,\mathbb{Z})$$**


### Currently available

1. Fourier expansion of holomorphic Siegel modular forms of degree $$g = 2$$

### Underway 

1. Fourier expansions for congruent subgroups
2. Fourier expansions for paramodular groups
3. Hecke theory
4. Borcherds lifts and Saito-Kurokawa lifts
5. Pull back to Hilbert modular forms

### Planned
 
1. Higher degree
2. Period polynomials
3. Non-holomorphic Siegel modular forms of degree 2
