# siegelmodularforms
A Pari/GP Package for Siegel Modular forms


### Currently available

1. Fourier expansion of holomorphic Siegel modualr forms of degree 2 for Sp(2,Z) 

### Underway 

1. Fourier expansions for congruent subgroups
2. Fourier expansions for paramodular groups
3. Hecke theory
4. Borcherds lifts and Saito-Kurokawa lifts
5. Pull back to Hilbert modular forms

### Planned
 
1. Higher degree
